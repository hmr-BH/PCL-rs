PCL-rs 是参考PCL用户界面，使用Rust+GPUI独立开发的社区版本，同时支持Windows、MacOS和Linux平台。

此版本的版本号与PCL官方并非对应关系，也请不要向官方仓库反馈社区版问题。

## 许可证

由于本项目属于《PCL 存储库合理使用指南》中规定的**重度使用**，因此采用与[PCL](https://github.com/Meloong-Git/PCL)相同的[许可证](./LICENCE.txt)。

项目发起人承诺，完全遵守《PCL 分发有限许可》和《PCL 存储库合理使用指南》中规定的全部内容。
